## Vue d'ensemble

### À propos de Stora

**Stora** est une plateforme SaaS moderne de gestion de self-storage fondée en **2020** à Banbridge, Royaume-Uni. La plateforme a été créée par **Gavin Shields**, CEO, qui possédait lui-même une installation de self-storage automatisée et a développé Stora en réponse aux frustrations avec les logiciels existants sur le marché.

### Reconnaissance et récompenses

- **Note Capterra:** 4.8/5
- **SSA Innovation Award 2024** (pour les cartes de sites nouvelle génération)
- **Membre officiel** de plusieurs associations internationales de self-storage
- **Disponibilité:** 46 pays, avec paiements acceptés depuis partout dans le monde
- **Levée de fonds:** Plus de 1,1M$ pour stimuler l'innovation dans le self-storage

### Philosophie produit

Stora se positionne comme **"le logiciel self-storage qui automatise l'administration pour que vous puissiez vous concentrer sur la croissance de votre entreprise."** La plateforme remplace les processus manuels par une solution unique et puissante qui gère les réservations, les paiements, l'accès intelligent et les données de performance en temps réel.

**Promesse clé:** Réduction jusqu'à **75% des tâches manuelles** grâce à l'automatisation de bout en bout.

// Model LeadScore / CustomerScore (déjà créé mais à améliorer)

public function calculateChurnRisk(): float {
    $score = 0;

    // Paiements en retard récents
    if ($this->hasLatePaymentsInLast3Months()) {
        $score += 30;
    }

    // Pas de connexion portail depuis 60 jours
    if ($this->lastPortalLogin < now()->subDays(60)) {
        $score += 20;
    }

    // Contacts support négatifs
    if ($this->hasNegativeSupportInteractions()) {
        $score += 25;
    }

    // Durée de location courte (< 6 mois)
    if ($this->contractDurationMonths < 6) {
        $score += 15;
    }

    // Saisonnalité (étudiant en fin d'année scolaire)
    if ($this->isStudentAndEndOfSchoolYear()) {
        $score += 10;
    }

    return min($score, 100); // 0-100 score
}

// Job quotidien
public function identifyChurnRisks() {
    $atRiskCustomers = Customer::active()
        ->get()
        ->filter(fn($c) => $c->calculateChurnRisk() > 60);

        // Déclencher campagne de rétention

#### C. Optimisation automatique des prix par box

**Au-delà du dynamic pricing global:**

**Logique avancée:**
```php
// Service AdvancedPricingService

public function optimizePricePerBox(Box $box) {
    $factors = [
        'occupancy_rate' => $this->site->current_occupancy_rate,
        'box_popularity' => $this->getBoxPopularityScore($box),
        'seasonal_demand' => $this->getSeasonalDemandMultiplier(),
        'competitor_pricing' => $this->getCompetitorPricing($box->size),
        'days_vacant' => $box->days_vacant,
        'location_premium' => $box->location_premium_factor,
    ];

    $basePrice = $box->base_price;

    // Algorithme de pricing
    $optimizedPrice = $basePrice;

    // Augmenter si forte demande
    if ($factors['occupancy_rate'] > 0.85) {
        $optimizedPrice *= 1.15;
    }

    // Diminuer si vacant longtemps
    if ($factors['days_vacant'] > 30) {
        $optimizedPrice *= 0.90;
    }

    // Ajuster selon saisonnalité
    $optimizedPrice *= $factors['seasonal_demand'];

    // S'aligner ou battre concurrence
    if ($factors['competitor_pricing'] < $optimizedPrice * 0.95) {
        $optimizedPrice = $factors['competitor_pricing'] * 0.98;
    }

    return round($optimizedPrice, 2);
}
```
